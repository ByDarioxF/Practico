## Tecnologías Utilizadas
El proyecto cumple estrictamente con las restricciones de la rúbrica:
* **HTML5:** Estructura semántica avanzada (`<header>`, `<footer>`, `<main>`, `<article>`, `<figure>`, `<details>`).
* **CSS3 Puro:** Utilizado en 5 páginas (`index.html`, `nosotros.html`, `equipo.html`, `galeria.html`, `faq.html`) garantizando coherencia visual global sin usar JS.
* **Bootstrap 5 (vía CDN):** Utilizado exclusivamente en 2 páginas (`catalogo.html` y `contacto.html`) implementando grid, cards, badges, alertas y formularios.
* **Docker:** Para la contenedorización del servidor web utilizando Nginx.
